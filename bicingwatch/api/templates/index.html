This site is used as a test for statistics I am using for my bicingwatch blog.

I am also making the data available as XML, JSON and text. If you are interested in this give me a shout.

The following pages are currently the only useful ones for the general public.

List of all bicing stations
A list of all current stations, each entry links to a detail page which gives information about location and usage statistics of that station
Map of all bicing stations
A google map of all the current station, each station links again to the detail page. The map is also bigger and quicker than the official map on bicing.com. You might have to zoom all the way in to see all the stations in one area.

Each page also contains a disqus comment area, if you like to add more information or comments to a station or this website.
